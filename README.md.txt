# Productivity Tracker Chrome Extension

A Chrome extension that tracks time spent on websites, classifies them as productive or unproductive, and displays weekly productivity analytics using a dashboard.

## Features
- Tracks active website time
- Automatically classifies websites
- Stores data in MongoDB
- Weekly productivity report
- Interactive dashboard using Chart.js

## Tech Stack
- Chrome Extension (JavaScript)
- Node.js + Express
- MongoDB
- Chart.js

## How to Run

### Backend
```bash
cd backend
npm install
node server.js
