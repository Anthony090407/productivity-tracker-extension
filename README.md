# Productivity Tracker Extension

A Chrome extension to track productivity while browsing the web.  

## Features
- Tracks time spent on websites
- Categorizes websites as productive/unproductive
- Dashboard to visualize productivity data
- Lightweight and easy-to-use

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js
- **Database:** MongoDB
